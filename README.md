# douyin-adaptation
原生Canvas开发的抖音小游戏可以用该skill一键生成，一遍过，不报错，同时接入侧边栏复访功能、广告系统等必接内容，非原生Canvas开发的游戏亦可使用，但效果可能欠佳。
注意！该skill的功能只有适配抖音小游戏，并不能决定游戏的生成质量，游戏的质量决定于你的想法和提示词，建议只在适配抖音小游戏的时候使用，其余的时候不要使用。
下面分享一些我在开发游戏的过程中经常用到的一些网站：
1.视频转序列帧：https://video2timesheet.dawnwindstudio.top/
2.抠图网址：https://www.koukoutu.com/removebgtool/all
3.图集网址：https://www.codeandweb.com/free-sprite-sheet-packer
4.体积减小网址：https://tinypng.com/
