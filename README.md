# douyin-adaptation

## 项目简介

原生Canvas开发的抖音小游戏可以用该skill一键生成，一遍过，不报错，同时接入侧边栏复访功能、广告系统等必接内容。

非原生Canvas开发的游戏亦可使用，但效果可能欠佳。

## ⚠️ 使用注意事项

该skill的功能只有**适配抖音小游戏**，并不能决定游戏的生成质量。

游戏的质量决定于你的想法和提示词，建议只在适配抖音小游戏的时候使用。
